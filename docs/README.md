# GymFlex - Gym Management System

A comprehensive web-based gym management system built with HTML, CSS, JavaScript, and Firebase.

## 🚀 Project Overview

GymFlex is designed to solve common problems in gym management:
- Digital receipt management (eliminates paper receipts)
- Automated member notifications
- Streamlined billing and membership management
- Multi-role access control (Admin, Member, User)

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Firebase (Firestore, Authentication, Storage)
- **Styling**: Custom CSS with modern design principles
- **Icons**: Font Awesome
- **Fonts**: Google Fonts (Inter)
